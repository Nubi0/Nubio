## 📌 프로젝트 소개

### 프로젝트 기획 배경
오송지하차도 침수 사고, 늘어나는 칼부림 등 자연재해나 인재와같은 재난 상황이 발생 하더라도 실시간 현황과 대처 방법을 알지 못해 피해가 발생합니다. <br/>
이러한 피해를 줄이고자 재난 발생 현황과 발생 시 대처할 수 있는 솔루션을 제공하고자 '맞춤형 안전 지도 공유 서비스' 프로젝트를 기획하게 되었습니다.


## 🔎주요 기능 설명
- 취향을 바탕으로 맞춤형 코스 추천
- 코스 커스텀 및 공유
- 재난상활 별 재난 솔루션
- 위험구역 제보 및 안전한 길 추천
